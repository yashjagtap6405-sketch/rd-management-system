# RD Management System

A full-stack Recurring Deposit (RD) management platform inspired by banking workflows.

This system allows users to create RD accounts, track deposits, calculate maturity payout, and check loan eligibility.

---

## Tech Stack

Backend
- Spring Boot
- Spring Data JPA
- PostgreSQL
- REST APIs
- Swagger (OpenAPI)

Frontend
- React
- Material UI
- Axios
- React Router

Database
- PostgreSQL relational model

---

## Features

### RD Account Management
- Create RD account
- Track monthly deposits
- Transaction history

### Deposit System
- One deposit allowed per month
- Penalty handling

### Maturity System
- Maturity payout calculation
- RD cycle restart after maturity

### Loan System
- Loan eligibility check
- Loan range calculation (60%–80%)
